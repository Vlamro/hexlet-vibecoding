hexlet-vibecoding

Простой одностраничный калькулятор индекса массы тела (ИМТ), дневной нормы калорий и нормы воды.

Откройте `index.html` в браузере, укажите рост, вес, возраст, пол и уровень активности — калькулятор покажет:
- ИМТ с текстовой категорией и короткой подсказкой;
- дневную норму калорий (формула Миффлина-Сан Жеора с учётом активности);
- дневную норму воды по весу.
